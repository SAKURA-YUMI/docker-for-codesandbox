# codesandbox.io

以codesandbox.io为例说明本docker使用

把Dockerfile和tasks.json上传，设置变量，启动即可，

其他容器有些只需Dockerfile(如keyob、报脸)，有些只需要Dockerfile里面的镜像地(如patr)

支持官网直连，固定隧道，临时隧道

支持订阅方式，url方式导入节点,日志里面有使用方法

变量只需设置PW即可

7860网址/vls 

7860网址/vms 

其他变量设置参考即可，https://github.com/dsadsadsss/java-wanju.git

这个是他的docker化
